# dotfiles

for using my dotfile first:

```sh
git clone https://github.com/booo2233/dotfiles.git
cd dotfiles
```
install GNU stow:

on arch
```sh
pacman -S stow
```
or
```sh
yay -S stow
```
on debian
```sh
apt update
apt install stow
```

to add all the dotfile do:
```sh
show .
```
for one specific configuration do:
```sh
show .tmux.conf
```
